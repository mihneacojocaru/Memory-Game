Memory Game

Paare gleicher, verdeckt aufliegender Kärtchen müssen durch Aufdecken im Wechsel der Spieler erkannt werden.

Dieses Spiel trainiert deine Konzentration, Ausdauer und Geduld und das visuelle Gedächtnis. Videospiele dagegen, wie sie oftmals auf der Playstation oder im PC-Bereich zu finden sind, verleiten oft eher dazu, dass jedes Ziel möglichst unmittelbar erreichbar sein müsste. Hilfreicher ist das Spiele Memory, das wichtige Elementarfähigkeiten trainiert. Spiele wie das Memory Spiel unterstützen vor allem auch Kinder bei deren Entwicklungsprozessen.

Die beste Denkleistung holst du aber aus dir heraus, wenn du es schaffst beide Gehirnhälften intelligent miteinander zu kombinieren.
